# A Dynamic Assessment Protocol (DAP) Prototype

This repository contains a functional prototype of the **Dynamic Assessment Protocol (DAP)** and its core instrument, the **Weighted Authorship Matrix (WAM)**. It was created as a tangible deliverable for the Master's Thesis proposal titled: *A Dynamic Protocol (DAP) Design for Signal Clarity in Generative AI Education: An HCI-Ethical Approach*.

## Purpose

The DAP is a socio-technical system designed to address the "Socio-Technical Evaluation Gap" in higher education assessments involving Generative AI. This prototype serves as an interactive demonstration of the WAM builder, an interface for instructors.

The key HCI principle demonstrated is the use of a **constrained user interface** to enforce the rules of constructive alignment and procedural justice, preventing the creation of pedagogically flawed or unfair assessment rubrics.
